About This Project

StudyMate helps students manage their schedule, finances, and studies efficiently. Built using SwiftUI and CoreData, it integrates an AI assistant for personalized study planning.

Tech Stack

  • Swift & SwiftUI
  • CoreData for persistence
  • Custom data models for transactions & courses
  • AI helper using OpenAI API (planned feature)
