Anagrams

A fun SwiftUI-based word puzzle game

Overview

Anagrams is a SwiftUI game where players form valid words from a random root word. The app includes live scoring logic and beautiful animations built for iOS.

Features

  • Dynamic scoring system for valid words
  • Real-time UI animations using SwiftUI
  • Fun educational gameplay
